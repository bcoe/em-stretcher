# Em::Stretcher

An EventMachine port of [Stretcher](https://github.com/PoseBiz/stretcher), a Fast, Elegant, ElasticSearch client

## Installation

Add this line to your application's Gemfile:

    gem 'em-stretcher'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install em-stretcher
